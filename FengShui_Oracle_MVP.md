# 風生水起 · FengShui Oracle
## MVP 產品藍圖 v1.0

---

## 產品定位

**一句話定位：** 最易用嘅中式命理 App，唔需要識睇盤，AI 幫你解釋。

**目標用戶：** 18-40歲大中華用戶，對命理有興趣但唔係專業人士。

**平台：** iOS + Android（第一版先做 Web App 測試）

---

## 變現模式（Freemium）

| 層級 | 內容 | 價格 |
|------|------|------|
| 免費 | 八字四柱、基本五行分析、求簽（每日1次）、基本籤文解釋 | $0 |
| 月費訂閱 | 深度命盤分析、流年運程、無限求簽、深度解簽 | HKD$28/月 |
| 單次付費 | 特定問題深度報告（轉工、感情、財運）| HKD$68/次 |

---

## MVP 功能列表

### 模組一：八字命盤（核心功能）
- [ ] 用戶輸入西曆出生年月日時
- [ ] 系統自動轉換農曆 + 計算天干地支
- [ ] 顯示四柱（年柱、月柱、日柱、時柱）
- [ ] 五行分佈圖（金木水火土）
- [ ] **免費：** 基本命格說明（例如：日主天干係甲木，代表...）
- [ ] **付費：** 深度分析（神煞、格局、用神、忌神）
- [ ] **付費：** 流年運程分析

### 模組二：求簽解簽
- [ ] 用戶選擇神仙（月老、觀音、關帝、黃大仙、王母娘娘、財神、媽祖）
- [ ] 搖簽筒動畫（手機傾斜或按掣）
- [ ] 隨機抽出1-100號籤
- [ ] 顯示籤文（文言文原文）
- [ ] **免費：** 基本籤義（一句話）
- [ ] **付費：** AI 深度解籤（針對感情/事業/健康/財運）

### 模組三：神明曆（日曆系統）
- [ ] 農曆日曆
- [ ] 神明生日提醒（觀音誕、關帝誕、王母娘娘誕等）
- [ ] 拜神指南（去邊間廟、買咩祭品、拜神流程）
- [ ] 吉凶日提示

---

## 技術架構

**前端：** React Native（iOS + Android 一次過）
**後端：** Node.js + Express
**AI：** Anthropic Claude API（解盤、解籤）
**資料庫：** Firebase（用戶資料、付費狀態）
**付費：** Stripe（信用卡）+ Apple Pay + Google Pay

---

## 開發優先順序

**Phase 1（MVP，目標6週）**

**Phase 3（上架後）**

2. 擇日系統
3. 風水羅盤
4. 合婚配對
